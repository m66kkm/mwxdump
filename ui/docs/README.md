### 设计团队
- 参考UI设计规范文档进行界面设计
- 遵循既定的设计系统和组件规范
- 确保设计一致性和用户体验质量

### 开发团队
- 严格按照设计规范实现UI组件
- 参考技术实现建议选择合适的技术栈
- 确保代码质量和可维护性

### 产品团队
- 了解界面功能布局和用户流程
- 参与用户体验优化和功能改进
- 协调设计和开发资源
